# ISL Word Detector

Indian Sign Language (ISL) Word Detection and Recognition system using Computer Vision and Machine Learning.

This project detects and recognizes Indian Sign Language hand gestures and translates them into meaningful words using feature extraction and classification techniques.

---

# Features

- Machine Learning based classification
- SIFT feature extraction
- KNN classification
- Scalable preprocessing pipeline
- Simple Python implementation
- Easy to extend for more gestures

---

# Tech Stack

- Python
- OpenCV
- Scikit-learn
- NumPy
- Gradio
- SIFT (Scale-Invariant Feature Transform)
- KMeans Clustering
- KNN Classifier
